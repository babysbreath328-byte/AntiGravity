# 百合子 ブログ構成（Yuriko Blog Structure）スキルガイド

このワークフロー（スキル）は、ご指定の月や季節の美容・肌トラブルに合わせ、エステサロン Lily garden の「百合子構文」を活かした**ブログ記事のタイトル案と構成（5記事分）**を自動生成するものです。

## 1. コンセプトと条件
- **対象サロン**: Lily garden（リリーガーデン）
- **メインテーマ**: スキンケア、肌質改善、毛穴ケア、ニキビケアなどの根本改善。
- **季節性**: 実行時点の月・季節における「よくある肌悩み（乾燥、花粉、紫外線、気温差、皮脂など）」をテーマに設定する。
- **構成条件**:
  - 全5記事のタイトル案と詳しい構成（見出しレベルでの流れ）を出力する。
  - そのうち **2〜3記事は「続き物（明日へ続く…の連作形式）」** とし、読者の期待感やブログへの再訪を促す工夫を入れる。

## 2. 記事ごとの構成フォーマット（百合子構文を適用）
各記事の構成案には、以下の要素を含めて出力してください。

- **【タイトル案】**: キャッチーで、思わずクリックしたくなる「悩み共感型」または「プロの視点型」のタイトル。
- **【狙い・対象者】**: どんな悩みを抱える人に読んでほしいか。
- **【構成の流れ】**:
  1. **導入（共感）**: 季節特有の悩みの提示。
  2. **展開（真実の指摘）**: なぜそのトラブルが起きているのか？（やりがちなNGケア等）。
  3. **サロンの答え（本質的解決）**: Lily gardenのアプローチ（土台作り、毛穴洗浄等）。
  4. **結び（次回予告 または 行動喚起）**: 続き物の場合は「原因は〇〇にありました。明日はその改善ステップをお伝えしますね」等で次へ繋げる。単発記事の場合は来店案内。

## 3. 「続き物（シリーズ設定）」の黄金パターン
続き物の場合は、以下のような流れで記事を分割します。
- **第1回（問題提起と共感）**: 「なぜこの季節に肌が荒れるの？」という原因の深掘りと、「あなただけじゃない」という共感。（結び：次回、意外な原因をお伝えします）
- **第2回（NGケアと真実）**: 読者がやりがちな間違った自己流ケアの指摘と、本当に必要な土台作りの重要性。（結び：では具体的にどうすればいい？明日はサロンでの秘密のケアをご紹介します）
- **第3回（解決策と希望）**: サロンでの根本改善アプローチ（イオン導入や毛穴洗浄など）の紹介と、実際のお客様の変化。「必ず変われる」という強いメッセージ。

## 4. 実行プロンプト（AIへの指示）
ユーザーから「百合子 ブログ構成（＋月や季節）」と依頼された場合は、その季節特有の肌悩みを抽出し、上記のフォーマットに従って5記事分の構成案を作成してください。
