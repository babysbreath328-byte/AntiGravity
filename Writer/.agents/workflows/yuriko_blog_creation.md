# 百合子 ブログ作成（Yuriko Blog Creation）スキルガイド

このワークフローは、作成された「構成案」をもとに、百合子構文の特徴（共感・専門性・伴走・優しさ）を活かしつつ、配信先のメディア（Hotpepper Beauty、Instagramリール、TikTok、Threads）に最適化された長さとフォーマットで本文や台本を作成するためのガイドラインです。

## 1. 媒体別の最適化ルールと百合子構文の適用

### ① Hotpepper Beauty ブログ用
- **文字数**: 500〜1000文字程度
- **特徴**: 百合子構文の「基本形」。改行と空行（余白）を多く取り、読みやすさを徹底的に重視。「〜ますね」「〜ですよね」「〜んです」といった語りかけ口調。
- **構成要素**: 季節の挨拶・導入 → お客様の心の声（共感） → NGケアと真実の指摘 → 本質的な解決策（施術メニュー） → 励ましと次回予告・予約への誘導。

### ② Instagram リール（Reels）用
- **形式**: 「動画内で喋る台本」＋「投稿キャプション（文章）」
- **台本の特徴**: 最初の3〜5秒で思わず手を止める「フック」が必要。「これやってない？」などの問いかけから入り、視覚的に惹きつける。動画で音声として話しても違和感のない、少しテンポの良い百合子構文。
- **キャプションの特徴**: 動画を見返さなくても内容が伝わるまとめと、サロンの想い、個別相談への優しい誘導。

### ③ TikTok用
- **形式**: 「ショート動画台本（テロップ指示含む）」
- **特徴**: リールよりもさらにテンポ感と結論ファーストを重視。「エステティシャンが教える」「実は〇〇なんです」といった意外性と権威性のアピール。「〜だよ」「〜してない？」とし少し親しみやすいトーンにするが、専門家としての軸はブラさない。
- **アクション**: フォローやコメント（「当てはまった人コメントしてね」など）を促す。

### ④ Threads用
- **形式**: テキスト主体の投稿（200〜400文字程度）
- **特徴**: 日常のつぶやきや、ふとした気付きを本音でシェアするようなスタイル。「今日こんなお客さまがいてね…」「みんな勘違いしてるんだけど…」というテキストならではの深い共感。
- **トーン**: 読者との距離が極めて近い「独り言＋問いかけ」のバランス。最後は無理に予約に繋げず、「自分の肌と向き合ってみてね」と優しく背中を押す。

## 2. 実行プロンプト（AIへの指示）
ユーザーから「百合子 ブログ作成をして」と依頼された場合、構成案の内容を読み込み、上記4つのプラットフォームそれぞれのフォーマットに合わせた本文を生成してください。百合子構文の「お客様を決して否定しない」「正しく導く覚悟」を全ての媒体で貫いてください。
